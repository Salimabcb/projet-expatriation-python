# Analyse des pays les plus attractifs pour une expatriation française

## 1. Objectif du projet

Ce projet vise à identifier les pays les plus attractifs pour les Français souhaitant s’expatrier.
L’analyse repose sur plusieurs dimensions quantitatives et qualitatives : coût de la vie, salaires, qualité de vie, stabilité économique, ainsi que l’adéquation entre les compétences des actifs français et les besoins du marché international.

L’ensemble du code, du traitement des données et des visualisations est disponible dans le notebook principal du projet.


## 2. Jeu de données analysés

Les analyses s’appuient sur plusieurs bases de données complémentaires, permettant une vision multidimensionnelle de l’attractivité des pays.

### a. Coût de la vie

* Analyse comparative du coût de la vie selon pays et continents
* Harmonisation des noms des pays
* Étude par catégories (logement, alimentation, transport, etc.)
* Classements régionaux et globaux

### b. Salaires

* Analyse des salaires moyens en 2023
* Importation d’une base complémentaire contenant les salaires en 2013
* Comparaison 2013 vs 2023 pour mesurer l’évolution salariale
* Classements par pays et par continent

### c. Qualité de vie

* Analyse d’indices de bien-être et de satisfaction
* Classements des continents et pays les mieux notés
* Variables explorées : sécurité, santé, stabilité, satisfaction, environnement

### d. Opportunités économiques

* Analyse des taux de chômage
* Étude des emplois vulnérables pour évaluer la stabilité économique
* Comparaison de la résilience économique entre pays
* Mise en perspective par continents

### e. Compétences françaises vs demande internationale

* Identification des compétences clés des actifs français
* Analyse des compétences les plus recherchées dans les pays attractifs
* Mise en évidence des correspondances ou écarts entre offre (France) et demande (pays d’accueil)



## 3. Technologies utilisées

Le projet repose sur un environnement Python orienté analyse de données.

* Python
* Jupyter Notebook
* pandas
* numpy
* matplotlib
* seaborn

Travaux réalisés :

* Nettoyage des données
* Fusion et harmonisation de datasets
* Analyse descriptive multidimensionnelle
* Visualisations comparatives
* Génération de classements par indicateurs


## 4. Conclusion

Cette étude propose une démarche complète pour identifier les destinations les plus favorables à une expatriation française.
En combinant des données économiques, sociales, salariales et qualitatives, le projet offre une vision globale et cohérente des pays présentant les meilleures conditions d’installation.

Les résultats obtenus peuvent servir :

* de support à une réflexion personnelle sur un projet d’expatriation,
* de base pour des recommandations,
* ou d’appui pour des analyses plus approfondies dans un cadre académique ou professionnel.
